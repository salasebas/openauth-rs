## [0.3.0] - 2026-06-15

### Added

- `RustAuthActixWebExt` for mounting auth routes into Actix Web (`mount_at_base_path`,
  `mount_routes`, `handle`).
- `RustAuthActixWebOptions` for per-mount configuration.
- Parity integration tests against the Axum adapter contract.
